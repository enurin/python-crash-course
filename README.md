# Python Crash Course Study Workspace

This folder was rebuilt from the uploaded *Python Crash Course, 3rd Edition* PDF.
It currently keeps the core Python chapters, 1 through 11, organized by chapter
and separates chapter concepts from exercise files.

## Layout

```text
python_crash_course/
  README.md
  chapters/
    chapter_01/
      concepts.md
      exercises/
        exercise_01_01_python_org.md
        exercise_01_02_hello_world_typos.py
```

## How to Study

1. Open `chapters/chapter_XX/concepts.md`.
2. Work through the exercise files in order.
3. Run `.py` files with Python.
4. Use `.md` files for conceptual, project-based, documentation, or deployment exercises.

## Notes

The exercise instructions are summarized rather than copied. Conceptual exercises use Markdown guide files when a single standalone Python
file would be misleading.
